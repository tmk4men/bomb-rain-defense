# UFOスマッシュ！（UFO SMASH!）

空から襲来する **UFO** を、パチンコ（スリングショット）で撃ち落として
地上の **人々** を守るドット絵アクションゲームです。

> 🎮 **今すぐ遊ぶ → https://tmk4men.github.io/bomb-rain-defense/**

---

## 遊び方

- パチンコの玉を **ドラッグで引っぱって離す** と発射！（マウス・タッチ対応）
- 引いた方向と反対に、放物線を描いて玉が飛びます（軌道予測の点線が出ます）。
- 玉は **貫通** するので、並んだUFOをまとめて撃ち抜けます。
- 1発で複数倒すと **コンボ倍率（最大×5）** が乗ります。
- UFOが人に触れると人がやられます。**全員やられるとゲームオーバー。**

## UFOの種類

| 種類 | 内容 |
| --- | --- |
| 🛸 ノーマル | シルバーの円盤。100点。 |
| 🛸 スピード | 青い円盤。速く落ちる。150点。 |

時間が経つほど落下が速くなっていきます。上位3スコアが記録されます。

## 特長

- 依存ライブラリなしの **HTML5 Canvas + 素のJavaScript**。
- 画面サイズ・縦横比に合わせて解像度が変わる **レスポンシブ＆フルスクリーン対応**。
- 低解像度バッファをニアレストネイバー拡大した **ドット絵風** ビジュアル。
- **効果音** は Web Audio API によるシンセ生成、**BGM** 付き。
- **PWA対応**（ホーム画面に追加してアプリのように起動可能）。
- ハイスコア（上位3件）・BGM設定は `localStorage` に保存。

## ファイル構成

```
index.html            … 画面構成・UI・OGP・PWA設定
style.css             … ピクセル風UIスタイル（CRTスキャンライン付き）
game.js               … ゲーム本体（描画・物理・UFO処理・効果音・BGM）
manifest.webmanifest  … PWAマニフェスト
icon-192/512.png      … アプリアイコン
og-image.png          … SNS共有用プレビュー画像
bgm.mp3               … BGM
```
