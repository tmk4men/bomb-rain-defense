# ボムレイン・ディフェンス（BOMB RAIN DEFENSE）

空から降ってくる **ボムへい** を、パチンコ（スリングショット）で撃ち落として
**花ばたけ** を守るドット絵アクションゲームです。

> 🎮 **今すぐ遊ぶ → https://tmk4men.github.io/bomb-rain-defense/**

---

## 遊び方

- パチンコの玉を **ドラッグで引っぱって離す** と発射！（マウス・タッチ対応）
- 引いた方向と反対に、放物線を描いて玉が飛びます（軌道予測の点線が出ます）。
- 玉は **貫通** するので、縦に並んだ敵をまとめて撃ち抜けます。
- 1発で複数の敵を倒すと **コンボ倍率（最大×5）** が乗ります。
- 敵が花に触れると花が枯れます。**花が全部枯れるとゲームオーバー。**

## 敵の種類

| 種類 | 内容 |
| --- | --- |
| ⚫ ノーマル | 基本の敵。100点。 |
| 🔵 スピード | 速く落ちる。150点。 |
| ⚙ てつ甲 | 2発当てないと壊れない。300点。 |
| 🟡 ゴールド | レア。500点。よく揺れる。 |

時間が経つほど落下が速く・敵が硬くなっていきます。ハイスコアを目指そう！

## 特長

- 依存ライブラリなしの **HTML5 Canvas + 素のJavaScript**。
- 低解像度（160×214）バッファに描画し、`imageSmoothingEnabled = false` で
  ニアレストネイバー拡大した **ドット絵風** ビジュアル。
- **効果音** は Web Audio API によるシンセ生成（音声ファイル不要）。
- 日本語ドットフォントに [DotGothic16](https://fonts.google.com/specimen/DotGothic16)、
  見出しに [Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P) を使用。
- ハイスコアは `localStorage` に保存。

## ローカルで動かす

`index.html` をブラウザで開くだけで遊べます。

```

og-image.png  … SNS共有用プレビュー画像
```
