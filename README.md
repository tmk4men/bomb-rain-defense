# ボムレイン・ディフェンス（BOMB RAIN DEFENSE）

空から降ってくる **ボムへい** を、パチンコ（スリングショット）で撃ち落として
**花ばたけ** を守るドット絵アクションゲームです。

『スーパーマリオ64DS』のミニゲーム **「ボムへいたいさくせん」(Bob-omb Squad)** に
着想を得たオリジナル作品で、敵の種類やパワーアップアイテムを独自に追加しています。

> 🎮 **遊ぶ**: [GitHub Pages で公開中](#) （`Settings → Pages` を有効化すると下のURLで遊べます）

---

## 遊び方

- パチンコの玉を **ドラッグで引っぱって離す** と発射！（マウスでもタッチでもOK）
- 引いた方向と反対に、放物線を描いて玉が飛びます（軌道予測の点線が出ます）。
- 落ちてくる敵を撃ち落として、画面下の **4つの花** を守りましょう。
- 敵が花に触れると花が枯れます。**花が全部枯れるとゲームオーバー。**
- 1発で複数の敵を倒すと **コンボ倍率（最大×5）** が乗ります。

## 登場するもの

| 種類 | 内容 |
| --- | --- |
| ⚫ ノーマルボム | 基本の敵。100点。 |
| 🔵 スピードボム | 速く落ちる。150点。 |
| ⚙ てつ甲ボム | 2発当てないと壊れない。300点。 |
| 🟡 ゴールドボム | レア。500点。よく揺れる。 |
| 💗 ハート | 撃つと枯れた花を1つ復活（満タンなら +300点）。 |
| ⭐ スター | 一定時間、玉が大きく＆貫通する強化状態に。 |
| ❄ フリーズ | 一定時間、全体がスロー（落下が遅く）に。 |
| 💥 ドカン | 画面内の敵をまとめて爆破！ |

時間が経つほど落下が速く・敵が硬くなっていきます。ハイスコアを目指そう！

## 技術メモ

- 依存ライブラリなしの **HTML5 Canvas + 素のJavaScript**。
- 低解像度（160×214）のオフスクリーンバッファに描画し、
  `imageSmoothingEnabled = false` でニアレストネイバー拡大して **ドット絵風** に。
- 日本語ドットフォントに [DotGothic16](https://fonts.google.com/specimen/DotGothic16)、
  見出しに [Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P) を使用。
- ハイスコアは `localStorage` に保存。

## ローカルで動かす

ただ `index.html` をブラウザで開くだけで遊べます。
（Webフォント読み込みのためネット接続があると見た目が完全になります）
